# 🌌 NASA Panel

A web application that displays NASA's Astronomy Picture of the Day using API integration.

This project was created to practice API consumption, JavaScript, backend structure with Node.js, and deployment using Docker.

---

## 🚀 Features

- NASA APOD API integration
- Dynamic image and description rendering
- Responsive layout
- Backend with Node.js
- Docker support
- Environment variable configuration

---

## 🛠 Technologies Used

- JavaScript
- HTML5
- CSS3
- Node.js
- Express
- Docker
- Docker Compose

## ⚙️ Running Locally

```bash
git clone https://github.com/livansena/NASA-Panel.git

cd NASA-Panel

npm install

npm start
```

---

## 🔐 Environment Variables

Create a `.env` file in the root folder and add your NASA API key:

```env
NASA_API_KEY=your_api_key_here
```
